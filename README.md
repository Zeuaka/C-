"Условия"

Задача 1.
Необходимо реализовать метод таким образом, чтобы он возвращал модуль 

числа х (если оно было положительным, то таким и остается, если он было 

отрицательным – то необходимо вернуть его без знака минус)

|

АЛГОРИТМ РЕШЕНИЯ:

Если число больше нуля, метод возвращает сам число, иначе умножает его на -1

|

Задача 3.
Необходимо реализовать метод таким образом, чтобы он возвращал true, если 

число x делится нацело на 3 или 5. При этом, если оно делится и на 3, и на 5, то 

вернуть надо false.

|

АЛГОРИТМ РЕШЕНИЯ:

Метод проверяет слеудющее условие: Если сумма остатка от деления Х на 3 + остатка от деления Х на 5 != 0

и только один из остатков от деления (на 5 или на 3) равен 0

|

Задача 5.
Необходимо реализовать метод таким образом, чтобы он возвращал 

максимальное из трех полученных методом чисел.

|

АЛГОРИТМ РЕШЕНИЯ:

Чтобы избежать вложенных условий, алгоритм работает следующим образом:

Первое условие - Если первое число больше второго и третьего, метод возвращает это число

Второе условие - Если второе число больше третьего, возвращаем второе число

Возвращаем 3 число

|

Задача 7.
Необходимо реализовать метод таким образом, чтобы он возвращал сумму 

чисел x и y. Однако, если сумма попадает в диапазон от 10 до 19, то надо вернуть 

число 20.

|

АЛГОРИТМ РЕШЕНИЯ:

Метод возвращает 20, если сумма двух чисел в диапазоне от 11 до 19 (включительно)

иначе возвращает их сумму

|

Необходимо реализовать метод таким образом, чтобы он выводил на экран 

квадрат из символов ‘*’ размером х, у которого х символов в ряд и х символов в

|

Необходимо реализовать метод таким образом, чтобы он выводил на экран 

треугольник из символов ‘*’ у которого х символов в высоту, а количество 

символов в ряду совпадает с номером строки, при этом треугольник выровнен

Метод реализован с помощью двух циклов, где один вложен в другой. Оба цикла ограничены x

При каждой итерации внешнего цикла, s уменшается на 1, а строка пререносится на следуюущую

"Массивы"

В каждой задаче этого раздела, пользователю предлагается создавать массивы тремя различными способами, с помощью

отдельного метода.

Задача 1.
Необходимо реализовать метод таким образом, чтобы он возвращал индекс 

первого вхождения числа x в массив arr. Если число не входит в массив –

возвращается -1.

|

АЛГОРИТМ РЕШЕНИЯ:

С помощью цикла for осуществялется проход по всем элементам массива,

если, во время этого прохода, находится такой элемент массива, который равен х,

выводится его индекс

|

Задача 3.
Необходимо реализовать метод таким образом, чтобы он возвращал 

наибольшее по модулю (то есть без учета знака) значение массива arr.

|

АЛГОРИТМ РЕШЕНИЯ:

Задается начальное значение максимума, указывающее на первый элемент массива

С помощью цикла for осуществялется проход по всем элементам массива,

Если находится элемент, больший максимума по модулю (модуль числа вычисляется с помощью метода abs),

максимум перезаписывается на этот элемент.

В конце работы, метод возвращает значения максимума

|

Задача 5.
Необходимо реализовать метод таким образом, чтобы он возвращал новый 

массив, который будет содержать все элементы массива arr, однако в позицию 

pos будут вставлены значения массива ins

|

АЛГОРИТМ РЕШЕНИЯ:

Задается значение int counter = 0, оно будет ориентиром для добавления в новый массив.

Так же создается массив int[] mix = new int[arr.Length + ins.Length], количество ячеек в котором будет равно

количеству ячеек в двух исходных массивов вместе

Далее С помощью цикла for осуществялется проход по всем элементам первого массива,

и благодаря заданному counter, индекс финального массива не теряется,

Каждую итерацию цикла происходит присваивание mix[counter] = arr[i],

а counter увеличивается на 1.

Далее, такая же процедура происходит и со вторым массивом.

В конце работы метод возвращает итоговый массив mix

|

Задача 7.
Необходимо реализовать метод таким образом, чтобы он возвращал новый 

массив, в котором значения массива arr записаны задом наперед.

|

АЛГОРИТМ РЕШЕНИЯ:

Создается новый массив arr1, в котором количество ячеек равно количеству в исходном массиве arr.

Далее задается стартовая позиция для нового массива int j = 0.

После чего происходит обратный проход исходного массива с помощью цикла for

for (int i = arr.Length - 1; i >= 0; i--), где на каждой итерации происходит присвоение

arr1[j] = arr[i], а счетчик j увеличивается на 1.

|

Задача 9.
Необходимо реализовать метод таким образом, чтобы он возвращал новый 

массив, в котором записаны индексы всех вхождений числа x в массив arr

|

АЛГОРИТМ РЕШЕНИЯ:

Проходим по всему массиву arr с помощью цикла for

Считается, сколько раз элемент x встречается в массиве.

Результат сохраняется в переменную counter.

Создается новый массив newarr размером counter.

Снова проходим по исходному массиву

При нахождении элемента x записываем его индекс в newarr

Используем переменную lenght для навигации, каждую итерацию она будет увеличиваться на 1.

В конце работы, метод возвращает newarr.
